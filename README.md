# Multi Step Form 

a responsive multi step form made with react js. the project is designed to provide a smooth user experience with proper steps , navigation and responsiveness

## Demo URL 

```https://multi-step-form-ten-woad.vercel.app/```


## Features 

- fully responsive (all devices)
- step functionality
- summary page before confirmation

## Teck Stack

- React JS
- Goggle Font
- Tailwind CSS
- Java Script
